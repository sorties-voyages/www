DESCRIPTION
=======
	Adds a dynamic sitemap.xml file to your site for SEO.
	
	The sitemap is splitted in various sub-sitemaps indexed in a sitemapindex file, this allows lightweight crawling.

	Sitemas are generated by the fly for each request. 
		(5000 urls limit per subsitemap, ordered by updated date, limit will be configurable in future releases)	

	Each one of this sub-sitemaps will contain:
	- 1. A set of handwrited URLs, specified in the settings page
	- 2. A dinamic set of urls for some entity (Blogs, users, files, etc)
	- 3. A dinamic set of your custom entities ( Entities that are unique of your site )

	Check out the screenshots

	Sitemaps are formated with XSL for candy reading	
	
	I welcome your comments and suggestions



INSTALL
=======

	- Unzip the code to your mod directory
	- Enable the plugin in the admin panel
	- Configure and preview the sitemap (If blank sitemap is displayed, flush the cache)
		You can preview before publishng it in http://YOURSITE/auto_sitemap/index
	- Once configured, add the following lines to the end off your elgg .htaccess file:
		RewriteRule ^sitemap.xml$ auto_sitemap/index [L]
		RewriteRule ^sitemap.xsl$ mod/auto_sitemap/views/default/auto_sitemap/sitemap.xsl [L]
		RewriteRule ^sitemapindex.xsl$ mod/auto_sitemap/views/default/auto_sitemap/sitemapindex.xsl [L]
	- Submit http://YOURSITE/sitemap.xml to Google !!!
